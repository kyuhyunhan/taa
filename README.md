# taa — These Are Agents

A collection of Claude Code agents that operate on the [`tas`](https://github.com/kyuhyunhan/tas) project.

`taa` hosts agent definitions, skills, and slash commands; `tas` is the target codebase those agents work against. The relationship mirrors [`gbrain`](https://github.com/garrytan/gbrain) ↔ [`gstack`](https://github.com/garrytan/gstack).
